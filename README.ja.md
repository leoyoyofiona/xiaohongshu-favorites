# 小紅書コレクションナビ

[简体中文](./README.md) | [English](./README.en.md)

小紅書のお気に入りを、あとで本当に使える形に整理するための macOS デスクトップアプリです。

## 解決したい課題

- お気に入りが増え続けて、必要な時に見つからない
- 小紅書の標準UIは長期整理に向いていない
- 論文、ツール、教育、アイデアが混ざって再利用しにくい
- 原文や画像をまとめて保存するのが面倒

## 主な機能

- Chrome でログイン済みの小紅書お気に入りを同期
- 重複排除と自動分類
- 3カラムの見やすいデスクトップUI
- 既読、重要マーク、前の記事/次の記事移動
- 現在の記事の本文と画像をワンクリックで書き出し
- 配布用 `.dmg` の生成

## 同期方式

- 現在の同期は `Chrome` を利用します
- 先に Chrome で小紅書のお気に入りページを開いてください
- その後アプリに戻って同期を実行します
- 埋め込みWebよりも安定しやすく、リスク制御に引っかかりにくい方式です

## インストール

### `.dmg` からインストール

1. `小红书收藏导航.dmg` を開く
2. `小红书收藏导航.app` を `Applications` にドラッグ
3. 初回起動で macOS に止められた場合は、右クリックして `開く` を選ぶか、システム設定で許可してください

### ソースから起動

必要環境:

- macOS 15+
- Xcode Command Line Tools
- Swift 6.2

```bash
swift run XHSOrganizerApp
```

## 使い方

1. Chrome で小紅書のお気に入りページを開く
2. アプリを開く
3. `同步小红书` をクリック
4. `从当前 Chrome 收藏夹同步` をクリック

記事詳細では `下载原文` を押すと:

- 本文テキスト
- 元リンク
- 画像一式

を `Downloads/小红书收藏导出/` に書き出します。

## パッケージ作成

```bash
./scripts/build_dmg.sh
```

生成物:

- `dist/小红书收藏导航.app`
- `dist/小红书收藏导航.dmg`

## 制限事項

- 現在の同期は Chrome 依存です
- 一部の小紅書記事はWeb側制限により原文取得率が下がります
- 以前の不完全リンクで取り込んだデータは、再同期すると改善する場合があります
